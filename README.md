# 크라켄의 술통 — 웹 프로토타입

Meshy-6로 생성한 3D 에셋을 사용하는 브라우저 기반 2~6인 로컬·온라인 턴제 파티 게임입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 표시되는 로컬 주소를 열고, 술통 주변의 빛나는 슬롯을 눌러 무기를 꽂습니다. 게임 설정에서 2~6명의 이름, 목표 점수, 게임 모드를 고를 수 있습니다.

## 현재 범위

- Meshy-6 PBR 모델 10종: 오크통·드럼통·화약통·해적 선장·무기 6종
- 1,024px WebP 텍스처와 GLB 메시로 웹 최적화
- Three.js 기반 GLB 렌더링
- 16개 슬롯과 칼 삽입 애니메이션
- 오크통 곡면에 밀착된 칼 구멍과 조준 상태
- 칼의 당기기·찌르기·충격 흔들림 액션
- 선장 해적검·D가드 커틀러스·크라켄 단검과 고등어·당근·우산 6종 선택
- 대형 오크통·코발트 블루 산업용 드럼통·저주받은 화약통 전환
- 나무 파편·금속 불꽃·화약 연기와 통별 충격음
- 외눈 크라켄 선장 캐릭터의 통 위 팝업 애니메이션
- 긴장도, 가짜 경보, 속도에 따라 변하는 사운드·조명·진동
- 2~6인 이름/색상/턴 전환과 라운드 누적 점수
- 클래식·더블 크라켄·7초 속전속결·역전 모드
- 별도 시작 화면에서 인원·모드·목표 점수·술통을 정하는 온라인 서버 설정
- 6자리 참가 코드와 QR 초대 링크를 사용하는 정원제 2~6인 온라인 대기실
- 참가자 이름 등록과 준비/취소, 전원 준비 확인, 차례별 휴대폰 조작 권한
- 연결 종료와 재접속 처리
- 방장 화면이 게임 상태를 관리하는 PeerJS 기반 실시간 P2P 멀티플레이
- 데스크톱/모바일 반응형 HUD

## 배포

GitHub 저장소의 `main` 브랜치를 기준으로 Vercel과 Sites 운영 주소에 배포합니다.

## 온라인 방

1. 방장이 시작 화면에서 `온라인 서버 열기`를 누릅니다.
2. 참가 인원, 게임 모드, 승리 목표와 사용할 술통을 고른 뒤 서버를 엽니다.
3. 참가자는 QR 코드, 초대 링크 또는 6자리 참가 코드로 접속해 이름을 등록합니다.
4. 설정한 정원이 모두 입장한 뒤 각 참가자가 휴대폰에서 `준비 완료`를 누릅니다.
5. 전원이 준비되면 방장이 `게임 시작`을 눌러 3D 게임 화면으로 이동합니다.
6. 휴대폰은 자기 차례에만 무기와 구멍을 고르고 `칼 꽂기`를 실행할 수 있습니다.

현재 온라인 모드는 방장 화면이 게임 서버 역할을 하는 호스트 중심 P2P 구조입니다. 연결에는 인터넷과 PeerJS 공개 시그널링 서버가 필요하며, 게임 중에는 방장 화면을 계속 열어두어야 합니다.
