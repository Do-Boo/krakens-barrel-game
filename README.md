# 크라켄의 술통 — 웹 프로토타입

Meshy로 생성한 오크통 GLB를 사용하는 브라우저 기반 2인 로컬 턴제 프로토타입입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 표시되는 로컬 주소를 열고, 오크통 주변의 빛나는 슬롯을 눌러 칼을 꽂습니다. 라운드마다 하나의 슬롯이 무작위 패배 슬롯으로 지정됩니다.

## 현재 범위

- Meshy 6 오크통 PBR 모델 및 웹 최적화 텍스처
- Three.js 기반 GLB 렌더링
- 16개 슬롯과 칼 삽입 애니메이션
- 오크통 곡면에 밀착된 칼 구멍과 조준 상태
- 칼의 당기기·찌르기·충격 흔들림 액션
- 선장 해적검·D가드 커틀러스·크라켄 단검 3종 선택
- 대형 오크통과 코발트 블루 산업용 드럼통 전환
- 2인 로컬 턴 전환
- 무작위 패배 슬롯과 해적 점프 임시 연출
- 데스크톱/모바일 반응형 HUD

## 배포

GitHub 저장소와 GitHub Pages 플레이 링크로 게시합니다. Vite의 상대 base 경로를 사용해 GitHub Pages의 저장소 하위 경로에서도 에셋이 정상 로드되도록 구성되어 있습니다.
