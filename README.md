# 크라켄의 술통 — 웹 프로토타입

Meshy-6로 생성한 3D 에셋을 사용하는 브라우저 기반 2~6인 로컬 턴제 파티 게임입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 표시되는 로컬 주소를 열고, 술통 주변의 빛나는 슬롯을 눌러 무기를 꽂습니다. 게임 설정에서 2~6명의 이름, 목표 점수, 게임 모드를 고를 수 있습니다.

## 현재 범위

- Meshy-6 PBR 모델 10종: 오크통·드럼통·화약통·해적 선장·무기 6종
- 1,024px WebP 텍스처와 GLB 메시로 웹 최적화
- Three.js 기반 GLB 렌더링
- 16개 슬롯과 칼 삽입 애니메이션
- 오크통 곡면에 밀착된 칼 구멍과 조준 상태
- 칼의 당기기·찌르기·충격 흔들림 액션
- 선장 해적검·D가드 커틀러스·크라켄 단검과 고등어·당근·우산 6종 선택
- 대형 오크통·코발트 블루 산업용 드럼통·저주받은 화약통 전환
- 나무 파편·금속 불꽃·화약 연기와 통별 충격음
- 외눈 크라켄 선장 캐릭터의 통 위 팝업 애니메이션
- 긴장도, 가짜 경보, 속도에 따라 변하는 사운드·조명·진동
- 2~6인 이름/색상/턴 전환과 라운드 누적 점수
- 클래식·더블 크라켄·7초 속전속결·역전 모드
- QR 코드로 연결하는 휴대폰 P2P 원격 컨트롤러
- 데스크톱/모바일 반응형 HUD

## 배포

GitHub 저장소와 GitHub Pages 플레이 링크로 게시합니다. Vite의 상대 base 경로를 사용해 GitHub Pages의 저장소 하위 경로에서도 에셋이 정상 로드되도록 구성되어 있습니다.

## 휴대폰 컨트롤러

게임 화면의 `QR 컨트롤` 버튼으로 방을 만들고 휴대폰에서 QR 코드를 스캔합니다. 휴대폰에서 무기와 1~16번 구멍을 고른 뒤 `칼 꽂기`를 누르면 게임 화면으로 명령이 전달됩니다. 연결에는 인터넷이 필요하며 PeerJS의 공개 시그널링 서버를 사용합니다.
